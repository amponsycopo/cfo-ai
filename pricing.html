Pertanyaan Umum
Apakah ada uji coba gratis? +
Ya! Setiap akun baru mendapat 3 analisis gratis untuk mencoba semua fitur Findible sebelum memutuskan upgrade.
Bagaimana cara upgrade atau downgrade paket? +
Hubungi kami via WhatsApp. Upgrade berlaku langsung, downgrade berlaku di awal bulan berikutnya. Tidak ada penalti atau biaya tersembunyi.
Apakah data keuangan saya aman? +
Data keuangan Anda diproses langsung dan tidak disimpan permanen di server kami. Kami menggunakan enkripsi SSL dan infrastruktur Supabase (AWS) yang memenuhi standar keamanan enterprise. Data Anda tidak pernah dijual atau digunakan untuk iklan.
Format file apa yang didukung? +
Findible mendukung Excel (.xlsx, .xls), CSV, dan Google Sheets. Kami juga mendukung format ekspor dari software akuntansi populer seperti Accurate.
Bagaimana jika kredit saya habis? +
Kredit direset setiap awal bulan sesuai paket Anda. Jika habis di tengah bulan, Anda bisa membeli kredit tambahan atau upgrade ke paket yang lebih tinggi via WhatsApp.
