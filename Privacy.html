Findible ("kami") berkomitmen untuk melindungi privasi Anda. Kebijakan ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi informasi Anda saat menggunakan layanan Findible di findible.pro.

1

Informasi yang Kami Kumpulkan

Kami mengumpulkan informasi berikut untuk menyediakan layanan analisis keuangan:

  • Informasi Akun: Alamat email dan password (disimpan terenkripsi melalui Supabase Auth)
  • Data Keuangan: File laporan keuangan (XLSX, XLS, CSV) yang Anda unggah untuk dianalisis — data ini diproses secara langsung dan tidak disimpan permanen di server kami
  • Data Google Sheets: Jika Anda menghubungkan Google Sheets, kami mengakses data spreadsheet yang Anda pilih dengan izin spreadsheets.readonly (hanya baca, tidak bisa mengubah data Anda)
  • Data Penggunaan: Jumlah kredit yang digunakan, tanggal koneksi Google Sheets, dan preferensi laporan otomatis
  • Informasi Bisnis: Nama bisnis dan jenis industri yang Anda masukkan secara sukarela
2

Cara Kami Menggunakan Data

Data yang kami kumpulkan digunakan semata-mata untuk:

  • Menganalisis laporan keuangan Anda menggunakan AI (Claude oleh Anthropic) dan menghasilkan laporan insight bisnis
  • Mengirimkan laporan analisis otomatis ke email Anda sesuai jadwal yang Anda pilih
  • Mengelola akun dan saldo kredit Anda
  • Meningkatkan kualitas layanan Findible

🔒 Data keuangan Anda tidak pernah digunakan untuk melatih model AI, dijual kepada pihak ketiga, atau digunakan untuk tujuan iklan.

3

Integrasi Google Sheets

Saat Anda menghubungkan Google Sheets ke Findible, berlaku ketentuan berikut:

  • Izin yang diminta: Hanya spreadsheets.readonly — kami hanya dapat membaca spreadsheet yang Anda pilih, tidak dapat mengubah, menghapus, atau mengakses file Google lainnya
  • Data yang diakses: Hanya data dari sheet yang Anda pilih secara eksplisit
  • Token akses: Refresh token Google Anda disimpan terenkripsi di database kami (Supabase) dan hanya digunakan untuk mengambil data sheet secara otomatis sesuai jadwal yang Anda aktifkan
  • Pencabutan akses: Anda dapat mencabut akses Findible ke Google Sheets kapan saja melalui Google Account Settings → Security → Third-party apps

⚠️ Penggunaan informasi yang diterima dari Google API mematuhi Google API Services User Data Policy, termasuk persyaratan Limited Use.

Findible menggunakan data Google Sheets hanya untuk tujuan yang secara eksplisit diminta pengguna: menghasilkan laporan analisis keuangan. Data ini tidak digunakan untuk tujuan lain, tidak dibagikan kepada pihak ketiga, dan tidak digunakan untuk iklan atau pemasaran.

4

Penyimpanan & Keamanan Data

  • Database: Data akun disimpan di Supabase (infrastruktur AWS, region ap-southeast-1 / Singapore), dengan enkripsi at-rest dan in-transit
  • Transfer data: Semua komunikasi menggunakan HTTPS/TLS
  • API Keys: Kunci API pihak ketiga (Anthropic, Resend, Google) disimpan sebagai environment variable server-side dan tidak pernah terekspos ke browser
  • Data keuangan: File yang diunggah diproses langsung di server dan tidak disimpan permanen — data hanya ada selama sesi analisis berlangsung
5

Berbagi Data dengan Pihak Ketiga

Kami bekerja sama dengan layanan pihak ketiga terpercaya berikut:

  • Anthropic (Claude AI): Data keuangan dikirim ke Anthropic API untuk menghasilkan analisis. Data ini tunduk pada Kebijakan Privasi Anthropic. Anthropic tidak menggunakan data API untuk melatih model.
  • Supabase: Penyedia database dan autentikasi. Data tunduk pada Kebijakan Privasi Supabase.
  • Resend: Layanan pengiriman email laporan. Data tunduk pada Kebijakan Privasi Resend.
  • Vercel: Platform hosting. Data tunduk pada Kebijakan Privasi Vercel.

Kami tidak menjual, menyewakan, atau membagikan data pribadi Anda kepada pihak lain di luar yang disebutkan di atas.

6

Hak Pengguna

Anda memiliki hak berikut terkait data pribadi Anda:

  • Akses: Meminta informasi tentang data pribadi yang kami simpan
  • Koreksi: Memperbarui data yang tidak akurat
  • Penghapusan: Meminta penghapusan akun dan semua data terkait
  • Portabilitas: Meminta salinan data Anda dalam format yang dapat dibaca mesin
  • Pencabutan izin Google: Mencabut akses Google Sheets kapan saja tanpa mempengaruhi akun Findible Anda

Untuk menggunakan hak-hak ini, hubungi kami melalui email di bawah.

7

Retensi Data

  • Data akun: Disimpan selama akun aktif
  • Data keuangan yang diunggah: Tidak disimpan permanen — dihapus setelah analisis selesai
  • Token Google: Disimpan selama fitur laporan otomatis aktif; dihapus saat Anda menonaktifkan fitur atau menghapus akun
  • Setelah penghapusan akun: Semua data dihapus dalam 30 hari
8

Perubahan Kebijakan

Kami dapat memperbarui kebijakan ini sewaktu-waktu. Perubahan signifikan akan diberitahukan melalui email atau notifikasi di aplikasi minimal 14 hari sebelum berlaku.

Penggunaan Findible setelah tanggal efektif perubahan dianggap sebagai persetujuan terhadap kebijakan yang diperbarui.
